# X1 XEN

Tap-to-claim community token on the X1 network.

## Overview
X1 XEN is a fixed-supply token distributed through a public on-chain tap mechanism.

Each tap:
- Transfers XEN from the vault
- Sends a small creator fee in XNT
- Updates the leaderboard

## Supply
Total supply: 1,000,000,000 XEN

- 800M — public tap-to-claim vault
- 200M — ecosystem treasury
